Install Theme

To start you must have a working version of WordPress. Almost all major web hosts provides one click WordPress install and you can consult your host's help site for more information in this regard. For manual install you can consult WordPress Codex Installing WordPress.

Once you have a working version of WordPress, You need to download "All Files and Documentation" from themeforest download page and extract the downloaded zip to get various files including inspiry-candor.zip.

Install Theme Via WordPress Dashboard ( Recommended Way )

  1. Go to 'Appearance > Themes' section
  2. Click 'Add New' and select the 'Upload Theme' option
  3. Click 'Upload Theme'
  4. Choose the inspiry-candor.zip file and press 'Install Now'
  5. Once the theme is uploaded you need to activate it.
  6. After activation go to "Appearance > Customize" to play with theme's settings.

if you face any problem during upload through dashboard, please upload the theme using FTP as guided below.

Install Theme Via FTP ( Alternative Way )

  1. Access your hosting server using an ftp client like FileZilla
  2. Go to the 'wp-content/themes' folder of your WordPress installation
  3. Extract the inspiry-candor.zip file and put the inspiry-candor folder in 'wp-content/themes/' folder
  4. Go to 'WordPress Dashboard > Appearance > Themes' section to activate the theme
  5. After activation go to "Appearance > Customize" to play with theme's settings.

Install Child Theme ( Highly Recommended )

It is highly recommended to use child theme, So that you can easily update parent theme whenever a new update becomes available. A pre-built ready to use child theme is included in the main theme pack that you have downloaded from themeforest.

To install child theme, You need to upload the inspiry-candor-child.zip and activate it in the same way as you have uploaded and activated parent theme's inspiry-candor.zip.

Add Blog Post

Go to WordPress Admin > Posts > Add New
  1. Provide the post title
  2. Provide the post contents
  3. Add or select the related category
  4. Add or select the related tags
  5. Provide the featured image. The image should have minimum width of 1140px. The bigger size images will be cropped automatically.
  6. Publish the post once it is ready.

Add Full Width Page

Go to WordPress Admin > Pages > Add New
  1. Provide the page title
  2. Provide the page contents

  3. Select the Full Width Template from page attributes.
  4. Publish the page once it is ready.
  5. After publishing you need to add the newly created page in menu from WordPress Admin > Appearance > Menus

Add Contact Us Page

You can use any of the following plugins to add Contact Form to your page(s)

  1. Jetpack
  2. Ninja Forms
  3. Contact Form 7

    Contact form Using Ninja Forms Plugin

    Once you have installed and activated the Ninja Forms plugin, click add contact form button to insert contact form on the page.


    Contact form Using Contact Form 7 Plugin

    Once you have installed and activated the Contact Form 7 plugin, click add contact form button to insert contact form on the page.

  3. Publish the page once it is ready.
  4. After publishing you need to add the newly created page in menu from WordPress Admin > Appearance > Menus

Author Profile Widget

  1. Go to Appearance > Widgets and look for Candor - Author Profile widget.
  2. Use that widget in a sidebar. Select the author from the list.
  3. Now check out the front end of your site and you will have the Author Profile widget working for your site.

Recent Posts Widget

  1. Go to Appearance > Widgets and look for Candor - Recent Posts widget.
  2. Use that widget in a sidebar. Provide its title and number of posts to display.
  3. Now check out the front end of your site and you will have the Recent Posts widget working for your site.

Top Posts Widget

  1. Go to Appearance > Widgets and look for Candor - Top Posts widget.
  2. Use that widget in a sidebar. Provide its title and number of posts to display.
  3. Now check out the front end of your site and you will have the Top Posts widget working for your site.

How to set up Twitter Widget

  1. Install the Quick and Easy Tweets plugin.
  2. Now, you need the twitter App's consumer key, consumer secret, access token and access token secret for twitter authentication.
    To get all these keys, You need to register a new twitter app from following url
    After registering app and generating access token you will have api keys and access token as displayed below

  3. Now go to Appearance > Widgets and look for Quick and Easy Tweets widget.
  4. Provide title, twitter username, number of tweets, keys and save the widget settings.
  5. Now check out the front end of your site and you will have a twitter widget working for your site.

How to set up Instagram Widget

  1. Install the Simple Instagram plugin.
  2. Go to Appearance > Widgets and look for Simple Instagram Feed widget.
  3. Provide title, User ID, number of images and save the widget settings.
  4. Now check out the front end of your site and you will have a working Instagram widget for your site.

How to Modify Styles

There are various options available to modify styles in this theme.

  1. Quick CSS - Go to Appearance > Customize and navigate to Styles to add any quick css.

  2. Custom CSS in Child Theme - 'child-custom.css' file is provided in child theme to add major css changes.

    Note: It is highly recommended to use child theme for any changes that you want to make as this way you can update parent theme whenever an update becomes available. Simply copy a template file that you want to change to child theme after creating same directory structure and then modify it. For more info http://codex.wordpress.org/Child_Themes.

  3. Custom CSS in Parent Theme - if you are not using child theme then 'css/custom.css' file is provided in parent theme to add major css changes.
